# LibanCoin

LibanCoin est une application full-stack composée de deux parties :

- un **backend** Node.js
- un **frontend** React.js
- **base de données** MongoDB

## Prérequis

Assurez-vous d’avoir installé :

- Node.js
- npm

## Structure du projet

```
marketPlace/
├── Backend/
└── Frontend/
```

## Installation

Installez et exécuter les dépendances dans chaque dossier.

Backend
```
cd Backend
npm install
nodemon.cmd
```
Frontend
```
cd Frontend
npm install
npm start
```
